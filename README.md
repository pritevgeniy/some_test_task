Задание 1

Кассовый чек содержит несколько позиций:

Кроссовки 2000₽,

Шорты 1000₽,

Футболка 500₽


Покупателю может быть предоставлена скидка.
Вам необходимо распределить скидку на заказ по всем позициям в чеке.
Скидка может быть 2х видов: N рублей на весь чек или M% на весь чек.

Задание 2

Дано:

Пользователь - Зарегистрированный в системе участник

Акция - некое маркетинговое предложение, которые показывается всем пользователям входящим в аудиторию. 

Аудитория - Набор фильтров, по которым формируются списки пользователей, например:
по среднему чеку, дате последнего визита или дате рождения.

Состав аудитории рассчитывается динамически, т.к. например, дата последнего визита может поменяться и пользователь выйдет из аудитории.


Задача:
Вывести список всех акций, которые доступны пользователю
